條件判斷式

C語言的條件判斷式

if(condition){ code... }

條件判斷式結構的開頭,一般分為三層if(開頭) else if(其他條件) else(條件之外)

if

條件判斷式結構開頭,必須存在,否則不為一個符合規則的程式

else if

當判斷的條件不只一種時可以加上else if 不一定需要存在

else

當除了前述的條件外,其餘的皆為同一種處理方式,則用else

範例中條件結構的區塊並未用大括號包裹

在C語言中,若條件區塊的code僅有一行,則大括號可以省略

條件運算子(這邊只介紹經常使用的)

==(等於) <(小於) >(大於) <=(小於等於) >=(大於等於) &&(and) ||(or)
